# OASIS VideoEditor Engine

## Overview

Backend engine for timeline-based video editing and rendering.

## Features

- Timeline system (tracks, clips, layering)
- Rendering pipeline (transforms, effects, transitions, overlays)
- Keyframe animation support
- API layer for frontend integration
- JSON serialization for timeline

## Project Structure

```text
VideoEditor/
  core/
  api.py
```

## How to Run

```bash
python test_editor.py
```

## Notes

- Backend is designed to integrate with a frontend UI
- Use api.py as the entry point for interaction
